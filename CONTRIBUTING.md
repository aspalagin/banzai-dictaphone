# Участие в разработке

1. Создайте ветку от `main` с одной понятной задачей.
2. Не добавляйте токены, `.env`, APK, keystore, записи или транскрипты.
3. Перед pull request выполните `python3 -m unittest discover -s server/tests -v`
   и `bash -n android/build-apk.sh`.
4. Опишите изменение, способ проверки и любые изменения формата или настроек.

Изменения должны быть небольшими и сопровождаться тестом, когда это возможно.
